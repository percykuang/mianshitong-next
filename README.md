# 面试通 Next

面试通 Next 是面试通的新一代 AI 面试辅导产品，面向正在准备前端与编程面试的开发者，提供从日常答疑、简历表达到模拟面试的一体化练习体验。

## 产品定位

面试通 Next 希望成为你的专属 AI 面试官和面试助手。它不仅回答零散问题，也会围绕真实求职场景，帮助你梳理经历、打磨表达、模拟追问，并在连续对话中沉淀上下文。

## 核心能力

- **AI 面试问答**：围绕前端、工程化、项目经验、性能优化等方向进行针对性答疑。
- **模拟面试陪练**：通过多轮对话模拟真实面试，帮助用户练习回答结构和临场表达。
- **职业状态理解**：在聊天过程中识别用户目标、阶段和上下文，让后续回复更贴合当前场景。
- **模型可切换**：本地开发可使用 Ollama，生产环境默认接入 DeepSeek。
- **后台管理**：提供用户、会话和额度等基础运营管理能力。

## 适合人群

- 正在准备前端校招、社招或晋升面试的开发者
- 希望提升项目表达、技术问答和追问应对能力的候选人
- 需要低成本反复练习模拟面试的学习者

## 当前状态

项目已经支持：

- Web 主站聊天体验
- Admin 后台登录与管理
- PostgreSQL 数据持久化
- DeepSeek 生产模型调用
- Docker + GitHub Actions 自动部署

## 访问入口

生产环境可按实际部署域名访问：

- Web 主站：`https://<web-domain>`
- Admin 后台：`https://<admin-domain>`
